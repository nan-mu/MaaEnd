## ✨ 功能一览

- 🎮 **实时作战辅助**：终极挂机神器，彻底解放双手！
  - **自动剧情**：一键跳过对话、智能选分支、快速过剧情，爽就完事了！（当然你想慢慢看也行~）
  - **交互处理**：自动点击指引光标（老普之手专业户）、翻阅说明书、点点点继续、关闭档案语音，再也不用手忙脚乱啦！
  - **辅助战斗**：自动普攻、放技能、开大招、打连携，还能锁定目标集火，战斗从此变得轻松写意✨
  - **自动拾取**：看到闪光就捡，绝不放过任何掉落物和采集点，资源收集小能手就是你！
  - **便捷操作**：秒速传送确认、自动滑滑索，赶路效率 MAX！
- 🧩 **拼图大师**：AI 火眼金睛，各种谜题一秒破解！支持单次速通、循环刷题、演示模式（学习一下大佬怎么解题 😎）。
- 💰 **自动贸易**：跑商专家上线！智能识别利润空间，自动低买高卖，跨区贸易轻松赚差价，财富自由指日可待！
- 📐 **蓝图搬运工**：建筑党福音！批量导入蓝图，智能解析分享码，工业流水线说建就建，效率拉满！
- 📅 **日常自动化**：每日任务一条龙服务！自动领取奖励、清空百科红点、领生产手册、完成 Baker 任务，日常肝度大幅降低 🎉
- 💻 **多端适配**：支持 PC 端 (Win32 API) 与安卓端 (ADB)，无论你在哪个平台，我们都能陪你摸鱼~
